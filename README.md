Chat Web App
A simple real-time chat application built with Node.js, Express, and SQLite on the backend, and React on the frontend.
Tech Stack
Backend: Node.js, Express, SQLite
Frontend: React
Project Structure
chat-app/
├── backend/     # Express server, routes, database
└── frontend/    # React client app
Getting Started
Backend
cd backend
npm install
node server.js
Frontend
cd frontend
npm install
npm start
